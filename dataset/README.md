# Credit Card Approval Prediction

## Project Overview

The Credit Card Approval Prediction project is a Machine Learning application that predicts whether a credit card application will be approved or rejected based on the applicant's financial and personal details.

The project helps banks and financial institutions automate the credit card approval process, reducing manual effort and improving decision-making accuracy.

---

## Objectives

- Analyze applicant data.
- Perform data preprocessing and feature engineering.
- Train multiple machine learning models.
- Compare model performance.
- Select the best-performing model.
- Deploy the model using a Flask web application.

## How to Run the Project

1. Clone the repository.

2. Install the required libraries.

```bash
pip install -r requirements.txt
```

3. Run the Flask application.

```bash
python app.py
```

4. Open your browser and visit:

```
http://127.0.0.1:5000
```
